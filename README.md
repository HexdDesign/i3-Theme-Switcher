### —— ⊹ i3 KEYBINDS ⊹ ——

i3 Keybinds - They can be altered to your preferences in the config file:


_~/.config/i3/config_ 

   
- [Click to see i3's official site for more information about keybindings](https://i3wm.org/docs/userguide.html#_default_keybindings)


:star: Mod4: SUPER / WIN KEY  


   
**MOD KEY:**

   _› set $mod Mod4_


**KILL FOCUSED WINDOW:**

   _› $mod+q_


**FULLSCREEN TOGGLE:**

   _› $mod+f_ 


**CHANGE FOCUS (VIM - STYLE):**

━━ FOCUS LEFT:

   _› $mod+j_

   _› $mod+k_ 

━━ FOCUS UP:

   _› $mod+l_ 

━━ FOCUS RIGHT:

   _› $mod+semicolon_  


**CHANGE FOCUS (ARROW KEYS):**

   _› $mod+Left_  

━━ FOCUS DOWN:

   _› $mod+Down_

   _› $mod+Up_ 

━━ FOCUS RIGHT:

   _› $mod+Right_
